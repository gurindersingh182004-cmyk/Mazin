# Mazin

This a procedurally-disruptive maze game, your goal is to reach the "g" on the maze, but the maze keeps changing it self.

## Controls

- w -> move up 
- s -> move down 
- a -> move left
- d -> move right

- es -> to exit the game 

make sure to press ENTER after each move

## How to run

### clone the repo
``` bash 
git clone https://github.com/gurindersingh182004-cmyk/Mazin
cd Mazin
```
then:
``` bash 
python Mazin.py
```
or
``` bash 
python3 Mazin.py
```
